# AGENTS GUIDANCE FILE

This repository powers DoorDrill, an AI-powered sales training platform.

Core architectural areas:

-   voice pipeline (FastAPI WebSockets)
-   conversation engine
-   persona simulation system
-   scenario library
-   grading engine
-   training loop

When making changes:

-   prioritize simulation realism
-   maintain low latency
-   preserve training loop speed
-   ensure prompts remain deterministic
