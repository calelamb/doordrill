## Part 4 — APNs Key (iOS Push Notifications)

- [ ] Go to https://developer.apple.com → Account → Certificates, IDs & Profiles → Keys
- [ ] Click `+` → Name it "DoorDrill Push" → check **Apple Push Notifications service (APNs)**
- [ ] Download the `.p8` file — **you can only download it once**
- [ ] Note your **Key ID** (shown on the key page) and your **Team ID** (top-right of any developer portal page)
- [ ] Store the `.p8` file securely — do NOT commit it to git

---

## Part 5 — FCM (Android Push Notifications)

- [ ] Go to https://console.firebase.google.com → Create project → "DoorDrill"
- [ ] Skip Google Analytics (optional)
- [ ] In project settings → Add app → Android
- [ ] Package name: `com.doordrill.app`
- [ ] Download `google-services.json`
- [ ] Place it at `mobile/android/app/google-services.json`
- [ ] In Firebase Console → Project Settings → Cloud Messaging → copy **Server Key**

## Part 6 — EAS Build Setup

```bash
# Install EAS CLI if not already installed
npm install -g eas-cli

# Login to Expo account (create one at expo.dev if needed)
eas login

# From mobile/ directory
cd mobile
eas build:configure
# This creates eas.json — review it
```

Edit `mobile/eas.json` to add a production profile:
```json
{
  "cli": {
    "version": ">= 10.0.0"
  },
  "build": {
    "development": {
      "developmentClient": true,
      "distribution": "internal"
    },
    "preview": {
      "distribution": "internal"
    },
    "production": {
      "ios": {
        "distribution": "store"
      },
      "android": {
        "buildType": "app-bundle"
      }
    }
  },
  "submit": {
    "production": {}
  }
}
```

- [ ] Run `eas credentials` to let EAS manage:
  - iOS: provisioning profile + signing certificate (requires Apple Developer account active)
  - Android: upload keystore — **back up the keystore file it generates**
- [ ] Upload APNs key to Expo: `eas credentials` → iOS → Push Notifications → Add APNs Key → paste Key ID, Team ID, upload .p8 file
- [ ] Add FCM Server Key to Expo: `eas credentials` → Android → Push Notifications → Add FCM Key
